Calculates the stake ratio p and reward rate R(p).

The reward rate R(p) is defined as:

R(p) = {
    0.03 / sqrt(2 * 0.01),         if 0 <= p <= 0.01
    0.03 / sqrt(2 * p),            if 0.01 < p <= 1
}
