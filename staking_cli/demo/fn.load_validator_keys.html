fn load_validator_keys(
    val_index: u16,
    mnemonic_phrase: Option<&str>,
) -> Result<(BLSKeyPair, StateKeyPair, Keypair)>
Expand description

Load consensus, state, and x25519 keys for a validator.

If mnemonic_phrase is Some, derives all three keys from the mnemonic at index DEMO_VALIDATOR_START_INDEX + val_index, matching the Ethereum signer account index for the same validator. Otherwise reads each key from the corresponding ESPRESSO_DEMO_NODE_{STAKING,STATE,X25519}_PRIVATE_KEY_{val_index} environment variable.
